## Description du projet

Ce projet consiste à développer un système intelligent de trading adaptatif basé sur l'intelligence artificielle appliqué aux métaux précieux (XAU/USD et XAG/USD).

Le système analysera les données historiques du marché, les indicateurs techniques, la structure du marché (SMC) ainsi que les nouvelles économiques afin de générer des recommandations de trading et d'assister la prise de décision.

---

- Collecter et préparer les données financières
